Retro Terminal, Git Version Management, et al cool features

A super-cool super-real virtual terminal environment is baked-in. The terminal supports node, npm, yeoman, grunt, express, amongst its strong pleothera of dev tools. Furthermore, the built-in text editor support emacs-key bindings, and you can connect projects to Git, and deploy on Heroku. Commands to 'git add + save,' etc or baked-in to the text-editor, and anyway the terminal is always one click away.

FAQs

Excuse me, but how is this going to work?
Aside from HTML, CSS and JS code, all other code will be run server-side in a secure sandbox. Terminal access will be available by a web client connected by Socket.IO. For future versions, we are considering SSH connectivity.
When will Athena be released?
We intend for a public release by Christmas, and a soon-coming beta releasing.
But then what will be the pricing, eh?
2GB for free across a maximum of 10 projects, and 25GB for $12 covering an unlimited number of projects.
What's the money for?
Whilst we have betas, and the scaffolding is set out, we need to expand our virtual terminal, concot the Python sandbox, revamp the editor, and add live debugging to Node.js, amongst a hoarder other nontrivial furtherances on our roadmap.
